# Шаблон постановки задачи: «Составить план реализации»

Два варианта:
- **LITE** — если ты можешь только описать функционал словами. Модель сама копает код и заполняет архитектурные детали.
- **FULL** — если ты уже знаешь, какие файлы/слои затронуты, и хочешь жёстко зафиксировать рамки.

В обоих случаях в ответ модель пишет ТОЛЬКО план, без кода.
