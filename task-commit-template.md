# Задача: Подготовь коммит

Приоритет - качество. Никаких уступок в сторону скорости или простоты решений.

Задача:
Подготовить коммит для всех изменений текущей сессии.

Шаги:

1. Обновить VERSION (patch/minor по docs/VERSIONING.md)

2. Обновить CHANGELOG.md (новая секция вверху)

3. Обновить docs:
   - docs/DECISIONS.md — если изменилось поведение
   - docs/TASK_STATE.md — статусы задач
   - docs/DEV_LOG.md — дата, файлы, валидация, риски
   - docs/KNOWN_ISSUES.md — если баг найден/закрыт

4. Предложить commit message в формате conventional commits

5. НЕ КОММИТИТЬ. Остановиться и ждать.
